This project is a javascript minecraft client for 1.7.10 that connects to a proxy server and sends requests for it to make a minecraft client (using the famous npm package, minecraft-protocol). The server must have online mode off but features that allow a real Minecraft account to use this client might come soon. You can contribute by helping in our discord or contribute to our Github. I will most likely not make a bedrock version. Eaglercraft was an inspiration for this project but there are some differences. This runs 1.7.10, can interact with real 1.7.10 (cracked) servers with the limitations of needing to connect to a proxy server to connect to minecraft, and the game engine isn't the same (it appears that eaglercraft uses a javascript java runtime while this uses the canvas element)
